# Vue 2 Task List Demo

A minimal Vue 2 task list application.

## Quick Start

```bash
npm install
npm run serve
```

Opens at http://localhost:8082

## Stack

- Vue 2.7.14
- Vuex 3.6.2
- Vue CLI 5

## Structure

- **src/App.vue** - Root component
- **src/components/TaskList.vue** - Task list display
- **src/components/TaskInput.vue** - Task input form
- **src/store/index.js** - Vuex store
